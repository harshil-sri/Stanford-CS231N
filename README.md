## 📖 About This Repository

This repository contains my personal coursework for **Stanford's CS231n: Convolutional Neural Networks for Visual Recognition**. It serves as a central hub for my lecture notes, Jupyter Notebook (`.ipynb`) explorations, and completed assignments.

My current progress: **Starting at Lecture 5 (Image Classification with CNNs)**.
